void Timer0_Initialize ( void  )

Initialize timer0 peripheral.

Initializes the 8-bit timer (timer0), sets it's period to 1 millisecond and enables the overflow interrupt.

Parameters
none.
Returns
none.
Note
_XTAL_FREQ must be running at 16MHz clock.

void Timer0_ISR ( void  )

Timer0 ISR used for timed events.

Counts the time in g_msTicks, g_100msTicks, g_ButtonsTimer, g_UpButton and g_DownButton timers. Also triggers time events by setting some flags g_FlagADCPeriod g_FlagSSDBlink and g_HeaterLED timer flag.

Parameters
none.
Returns
none.
